# Snippetbox

## Routing request
| Method | URL Pattern | Handler    | Action    |
| :---:   | :---: | :---: | :---: |
| ANY | /   | home   | Display the home page   |
| ANY | /snippet/view    | snipperView   | Display a specific snippet   |
| POST | /snippet/create   | snippetCreate   | Create a new snippet   |
